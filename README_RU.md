# Scavenger

Этот мод добавляет _захватывающие_ **спидраны предметов** в Minecraft!

## Что такое спидран предметов?
![Таймер](https://cdn.modrinth.com/data/cached_images/7ad01e8cddc7545b81a5a1add5cb86fa63269c4c.png)

Когда вы создаёте мир — перед вами появляются несколько рулеток, которые можно прокрутить, чтобы определить вашу цель. После создания мира запускается таймер, и ваша задача — найти выпавший предмет, соблюдая дополнительный модификатор, который вам достался!

![Анимация рулетки](https://github.com/catboybinary/scavenger/blob/master/images/wheels.webp?raw=true)

## Настройка
Файл конфигурации находится по пути:
`config/scavenger.yaml`

Полный и актуальный список модификаторов можно найти в [файле локализации здесь](https://github.com/nrxzwq/scavenger-host/blob/main/common/src/main/resources/assets/scavenger/lang/en_us.json)

## И вперёд!
Мод имеет широкие возможности настройки и умеренную расширяемость. Также можно создавать простые дополнения, добавляющие новые модификаторы (хотя эта функция ещё не полностью протестирована и может измениться :P)
